# agent.cognition.v1

Run:
  bash install.sh --workspace YOUR_WORKSPACE_PATH

If unsure of workspace path:
  bash install.sh  (auto-detects from ~/.openclaw/openclaw.json)

Then restart your gateway:
  openclaw gateway restart

Done. Your agent is now running:
- MemBrain v2 — lossless memory with BM25 + semantic recall
- NervousSystem — nightly self-improvement loop
- ImmuneSystem — preflight protection gates
- 7 Cognitive Signatures — installable thinking patterns

To install a signature into your SOUL.md:
  node ~/.openclaw/extensions/signatures/install.js <signature-name>
  # Options: the-surgeon, the-locksmith, the-architect, the-auditor,
  #          the-weaver, the-refiner, the-cartographer
